💬 How comment fetching works (3 strategies tried in order):
1. Proxy /jira?apiPath=… — Most reliable. Add this snippet to your Cloudflare Worker's /jira handler.
2. Proxy /jira/comments?issueKey=… — Legacy endpoint.
3. Direct Jira Cloud API — Works for Atlassian Cloud automatically (no proxy needed for comments).
📋 Add apiPath support to your Cloudflare Worker 
// Add at the TOP of your /jira route handler (before the JQL search):
const apiPath = url.searchParams.get('apiPath');
if (apiPath) {
  const jiraBase2 = url.searchParams.get('jiraBase');
  const auth2 = url.searchParams.get('auth');
  const apiResp = await fetch(jiraBase2 + apiPath, {
    headers: { 'Authorization': 'Basic ' + auth2, 'Accept': 'application/json' }
  });
  const apiData = await apiResp.json();
  return new Response(JSON.stringify(apiData), { status: apiResp.status, headers: corsHeaders });
}

          ✏️ Add Comment to Jira
          
        

        

          ⚠️ Comments require your Cloudflare Worker to have a /jira/comment POST route.

          Add this to your Worker (cool-queen-c9f2.jirapoc.workers.dev):

          
if (url.pathname === '/jira/comment') {
  const { jiraBase, auth, issueKey, body } = await request.json();
  const r = await fetch(`${jiraBase}/rest/api/3/issue/${issueKey}/comment`, {
    method: 'POST',
    headers: { 'Authorization': 'Basic ' + auth,
                'Content-Type': 'application/json',
                'Accept': 'application/json' },
    body: JSON.stringify(body)
  });
  return new Response(await r.text(), { status: r.status, headers: corsHeaders });
}

        

        
        

          
          

            
            
          

        

      

    

    

      
      
    

  







  

    

      

        
💬 MS TEAMS INTEGRATION

        
Send Overdue Nudges via Teams

      

      
    

    

      

        📌 Webhook URL loaded from 🔌 Connect Jira. To get one: Teams Channel → ··· → Connectors → Incoming Webhook → Create → Copy URL

        

          📶 How delivery works:

          With Cloudflare proxy → add /teams-nudge to your Worker (see code below) → confirmed ✅ delivery.

          Without proxy → browser uses no-cors mode → message IS sent to Teams but browser can't confirm it (shows ✅ Sent*).

          

            📋 Add this to your Cloudflare Worker for confirmed delivery
            
if (url.pathname === '/teams-nudge') {
  const { webhookUrl, payload } = await request.json();
  const r = await fetch(webhookUrl, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(payload)
  });
  const txt = await r.text();
  return new Response(txt, { status: r.status, headers: corsHeaders });
}
